✏️ TextUtils – React Text Utilities App

TextUtils is a responsive and interactive web application built with React. It provides a collection of text-processing tools to modify, analyze, and interact with text efficiently. The app supports light/dark modes, real-time text analysis, and even speech synthesis.


- Convert text to uppercase or lowercase
- Text-to-speech functionality
- Dark/Light mode toggle for comfortable reading
- Real-time text summary (word count, character count, estimated reading time)
- Smooth alerts for actions
- Clean and responsive UI
- Built using React functional components and Hooks

Tech Stack
- React.js (Functional Components + Hooks)
- React Router for navigation
- Bootstrap 5 for styling
- JavaScript (ES6)
- HTML5 & CSS3
